# 🍽️ Zomato Data Analysis (Mumbai & Chennai)

## 📌 Project Overview

This project focuses on **Exploratory Data Analysis (EDA)** of Zomato restaurant datasets for **Mumbai** and **Chennai**.
The goal is to analyze restaurant trends, pricing, ratings, and customer preferences using Python and data visualization techniques.

---

## 🎯 Objectives

* Understand restaurant distribution across locations
* Analyze cuisine popularity
* Study relationship between price and ratings
* Identify affordable and reliable restaurants
* Compare patterns between Mumbai and Chennai

---

## 📂 Dataset Information

### 🔹 Mumbai Dataset

* Contains details like Region, Cuisine Type, Price, Rating, Votes, etc.

### 🔹 Chennai Dataset

* Contains details like Location, Cuisine, Price for 2, Dining Rating, Delivery Rating, Features, etc.

---

## 🧹 Data Preprocessing

* Handled missing values (NaN)
* Removed irrelevant records
* Converted data types to numeric format
* Cleaned text columns (Region, Cuisine, etc.)
* Standardized inconsistent values

---

## 📊 Exploratory Data Analysis (EDA)

### Key Analysis Performed:

1. Restaurant distribution by location/cuisine
2. Rating distribution analysis
3. Top-rated restaurants
4. Delivery service analysis
5. High-rated restaurant locations
6. Price distribution of highly rated restaurants
7. Cuisine-specific market analysis (e.g., Chinese)
8. Price vs Rating relationship
9. Location vs Price analysis
10. Affordable restaurants
11. Most reliable restaurants (based on rating, reviews, and cost)

---

## 📈 Tools & Technologies Used

* Python 
* Pandas
* NumPy
* Plotly (for interactive visualizations)
* Jupyter Notebook

---

## 💡 Key Insights

* Highly rated restaurants are mostly in the mid-price range
* Price does not strongly determine ratings
* Some locations have a higher concentration of top-rated restaurants
* Affordable restaurants can also provide high quality
* Customer reviews (rating count) play an important role in reliability

---

## 📌 Conclusion

This analysis helps in understanding customer preferences and restaurant trends, which can be useful for business decisions and market analysis.
